# Agno 文档中文化计划

## 计划概述
本计划旨在系统性地检查 `agno-docs-cn` 项目下所有目录中的文档文件，识别尚未中文化的内容，并按目录逐个完成中文化工作。

## 当前状态分析
- 项目包含大量 `.mdx` 格式的文档文件
- 部分文件（主要在 `basics\agents\usage\` 目录下）已经完成中文化
- 大部分文件仍为英文原文，需要翻译

## 执行步骤

### 1. 全面扫描和分类
- [ ] 扫描所有 `.mdx` 文件
- [ ] 识别已中文化文件（包含中文字符）
- [ ] 识别未中文化文件（纯英文）
- [ ] 按目录结构分组未中文化文件

### 2. 按目录优先级排序
- [ ] `basics\chat-history\*` - 聊天历史相关文档
- [ ] `basics\context\*` - 上下文相关文档
- [ ] `basics\dependencies\*` - 依赖管理相关文档
- [ ] `basics\evals\*` - 评估相关文档
- [ ] `basics\guardrails\*` - 防护栏相关文档
- [ ] `basics\hitl\*` - 人在回路相关文档
- [ ] 其他剩余目录

### 3. 逐目录中文化实施
对每个目录执行以下步骤：
- [ ] 阅读目录下所有未中文化文件
- [ ] 理解技术内容和上下文
- [ ] 进行准确的技术翻译
- [ ] 保持原有格式和代码块不变
- [ ] 验证翻译质量

### 4. 质量保证
- [ ] 确保技术术语一致性
- [ ] 保持文档结构完整性
- [ ] 验证所有链接和引用仍然有效

## 预期成果
- 所有文档文件完成高质量中文化
- 保持原有的技术准确性和文档结构
- 提供完整的中文版 Agno 文档

## 注意事项
- 严格保持原有代码块、配置示例不变
- 技术术语采用行业标准中文翻译
- 保留所有元数据（title, description 等）的英文原文，仅翻译正文内容
- 确保 Markdown/MDX 语法正确性

Agno 支持 using 本地 JSON files as a "数据库" with the `JsonDb` class.
This is a simple way to store your Agent's session 数据 without having to setup a 数据库.

<Warning>
Using JSON files as a 数据库 is not recommended for production applications.
Use it for demos, testing and any other use case where you don't want to setup a 数据库.
</Warning>

## 使用方法

```python json_for_agent.py
from agno.agent import Agent
from agno.db.json import JsonDb

# Setup the JSON database
db = JsonDb(db_path="tmp/json_db")

# Setup your Agent with the Database
agent = Agent(db=db)
```

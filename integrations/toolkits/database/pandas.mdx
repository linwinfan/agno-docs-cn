**PandasTools** enable an Agent to perform 数据 manipulation tasks using the Pandas library.

```python cookbook/14_tools/pandas_tool.py
from agno.agent import Agent
from agno.tools.pandas import PandasTools

# Create an agent with PandasTools
agent = Agent(tools=[PandasTools()])

# Example: Create a dataframe with sample data and get the first 5 rows
agent.print_response("""
Please perform these tasks:
1. Create a pandas dataframe named 'sales_data' using DataFrame() with this sample data:
   {'date': ['2023-01-01', '2023-01-02', '2023-01-03', '2023-01-04', '2023-01-05'],
    'product': ['Widget A', 'Widget B', 'Widget A', 'Widget C', 'Widget B'],
    'quantity': [10, 15, 8, 12, 20],
    'price': [9.99, 15.99, 9.99, 12.99, 15.99]}
2. Show me the first 5 rows of the sales_data dataframe
""")
```

## Toolkit 参数

| Parameter                        | Type   | Default | Description                                          |
| -------------------------------- | ------ | ------- | ---------------------------------------------------- |
| `enable_create_pandas_数据frame` | `bool` | `True`  | Enables functionality to create pandas DataFrames.  |
| `enable_run_数据frame_operation` | `bool` | `True`  | Enables functionality to run DataFrame operations.   |
| `all`                            | `bool` | `False` | Enables all functionality when set to True.         |

## Toolkit Functions

| Function                  | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `create_pandas_数据frame` | Creates a Pandas DataFrame named `数据frame_name` by using the specified function `create_using_function` with parameters `function_parameters`. Parameters include '数据frame_name' for the name of the DataFrame, 'create_using_function' for the function to create it (e.g., 'read_csv'), and 'function_parameters' for the arguments required by the function. Returns the name of the created DataFrame if successful, otherwise returns an error message. |
| `run_数据frame_operation` | Runs a specified operation `operation` on a DataFrame `数据frame_name` with the parameters `operation_parameters`. Parameters include '数据frame_name' for the DataFrame to operate on, 'operation' for the operation to perform (e.g., 'head', 'tail'), and 'operation_parameters' for the arguments required by the operation. Returns the result of the operation if successful, otherwise returns an error message.                                          |

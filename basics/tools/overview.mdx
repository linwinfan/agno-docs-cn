**工具是智能体与外部世界交互的接口。**

工具的核心概念：

- **功能：** 为智能体提供特定的能力
- **接口：** 标准化的方式让智能体使用外部服务
- **安全性：** 控制智能体可以执行的操作

## 工具类型

1. **内置工具：** Agno 提供的常用工具
2. **自定义工具：** 开发者创建的专用工具
3. **MCP 工具：** 通过 Model Context Protocol 连接的工具
4. **第三方工具：** 集成外部服务的工具

## 常见工具示例

- **搜索工具：** DuckDuckGo、Google 搜索
- **数据库工具：** SQL、NoSQL 数据库操作
- **API 工具：** REST API 调用
- **文件工具：** 文件读写和处理

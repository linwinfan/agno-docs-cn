---

当智能体遇到错误或速率限制时，团队会自动重试，并在每次尝试之间增加延迟时间。

<Steps>

  <Step title="创建 Python 文件">
    ```bash
    touch team_exponential_backoff.py
    ```
  </Step>

  <Step title="将以下代码添加到您的 Python 文件中">

    ```python team_exponential_backoff.py
    from agno.agent import Agent
    from agno.team import Team
    from agno.tools.duckduckgo import DuckDuckGoTools

    # 创建一个研究团队
    team = Team(
        members=[
            Agent(
                name="Sarah",
                role="数据研究员",
                tools=[DuckDuckGoTools()],
                instructions="专注于收集和分析数据",
            ),
            Agent(
                name="Mike",
                role="技术作者",
                instructions="创建清晰、简洁的摘要",
            ),
        ],
        retries=3,
        exponential_backoff=True,
    )

    team.print_response(
        "Search for latest news about the latest AI models",
        stream=True,
    )
    ```
  </Step>

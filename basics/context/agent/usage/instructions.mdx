---
title: 基本智能体指令
---

本示例演示了如何向智能体提供基本指令，以指导其响应行为和讲故事风格。

## 代码

```python instructions.py
from agno.agent import Agent

agent = Agent(instructions="分享一个关于")
agent.print_response("公元12000年的爱情的两句话故事。")
```

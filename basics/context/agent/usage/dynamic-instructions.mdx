---
title: 基于会话状态的动态指令
---

本示例演示了如何创建基于会话状态变化的动态指令，为不同用户提供个性化的智能体行为。

## 代码

```python dynamic_instructions.py
from agno.agent import Agent
from agno.run import RunContext

def get_instructions(run_context: RunContext):
    if not run_context.session_state:
        run_context.session_state = {}

    if run_context.session_state.get("current_user_id"):
        return f"编写一个关于 {run_context.session_state.get('current_user_id')} 的故事。"

    return "编写一个关于用户的故事。"


agent = Agent(instructions=get_instructions)
agent.print_response("写一个两句话的故事", user_id="john.doe")
```

---
title: 通过函数提供动态指令
---

本示例演示了如何通过函数向智能体提供指令，该函数可以访问智能体的属性，从而实现动态和个性化的指令生成。

## 代码

```python instructions_via_function.py
from typing import List

from agno.agent import Agent


def get_instructions(agent: Agent) -> List[str]:
    return [
        f"您的名字是 {agent.name}！",
        "用俳句说话！",
        "使用诗歌来回答问题。",
    ]


agent = Agent(
    name="AgentX",
    instructions=get_instructions,
    markdown=True,
)
agent.print_response("您是谁？", stream=True)
```

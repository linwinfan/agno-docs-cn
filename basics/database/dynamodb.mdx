Agno 支持使用 [DynamoDB](https://aws.amazon.com/dynamodb/) 作为数据库，通过 `DynamoDb` 类实现。

## 用法

要连接到 DynamoDB，您需要有效的 AWS 凭证。您可以将它们设置为环境变量：

- `AWS_REGION`: 要连接的 AWS 区域。
- `AWS_ACCESS_KEY_ID`: 您的 AWS 访问密钥 ID。
- `AWS_SECRET_ACCESS_KEY`: 您的 AWS 秘密访问密钥。

```python dynamo_for_agent.py
from agno.db.dynamo import DynamoDb

# 设置您的数据库
db = DynamoDb()

# 使用数据库设置您的代理
agent = Agent(db=db)
```

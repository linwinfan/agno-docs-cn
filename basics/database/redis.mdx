Agno 支持使用 [Redis](https://redis.io/) 作为数据库，通过 `RedisDb` 类实现。

## 用法

### 运行 Redis

安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 并使用以下命令在端口 **6379** 上运行 **Redis**：

```bash
docker run -d \
  --name my-redis \
  -p 6379:6379 \
  redis
```

```python redis_for_agent.py
from agno.agent import Agent
from agno.db.redis import RedisDb

# 初始化 Redis 数据库（为您的设置使用正确的 db_url）
db = RedisDb(db_url="redis://localhost:6379")

# 使用 Redis 数据库创建代理
agent = Agent(db=db)
```

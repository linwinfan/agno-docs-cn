Agno 支持使用 [MySQL](https://www.mysql.com/) 作为数据库，通过 `MySQLDb` 类实现。

## 用法

```python mysql_for_agent.py
from agno.agent import Agent
from agno.db.mysql import MySQLDb

# 设置您的数据库
db = MySQLDb(db_url="mysql+pymysql://ai:ai@localhost:3306/ai")

# 使用数据库设置您的代理
agent = Agent(db=db)
```

### 运行 MySQL

安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 并使用以下命令在端口 **3306** 上运行 **MySQL**：

```bash
docker run -d \
  --name mysql \
  -e MYSQL_ROOT_PASSWORD=ai \
  -e MYSQL_DATABASE=ai \
  -e MYSQL_USER=ai \
  -e MYSQL_PASSWORD=ai \
  -p 3306:3306 \
  -d mysql:8
```

Agno 支持异步使用 [Sqlite](https://www.sqlite.org)，通过 `AsyncSqliteDb` 类实现。

## 用法

```python async_sqlite_for_agent.py
from agno.agent import Agent
from agno.db.sqlite import AsyncSqliteDb

# 设置 SQLite 数据库
db = AsyncSqliteDb(db_file="tmp/data.db")

# 使用 SQLite 数据库设置基本代理
agent = Agent(db=db)
```

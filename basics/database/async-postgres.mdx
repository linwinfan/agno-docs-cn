Agno 支持异步使用 [PostgreSQL](https://www.postgresql.org/)，通过 `AsyncPostgresDb` 类实现。

## 用法

```python postgres_for_agent.py
from agno.agent import Agent
from agno.db.postgres import AsyncPostgresDb

# 替换为您自己的连接字符串，注意 `async_` 前缀
db_url = "postgresql+psycopg_async://ai:ai@localhost:5532/ai"

# 设置您的数据库
db = AsyncPostgresDb(db_url=db_url)

# 使用数据库设置您的代理
agent = Agent(db=db)
```

### 运行 Postgres（带 PgVector）

安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 并使用以下命令在端口 **5532** 上运行 **PgVector**：

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agnohq/pgvector:16
```

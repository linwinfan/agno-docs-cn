Agno 支持使用 [Sqlite](https://www.sqlite.org) 作为数据库，通过 `SqliteDb` 类实现。

## 用法

```python sqlite_for_agent.py
from agno.agent import Agent
from agno.db.sqlite import SqliteDb

# 设置 SQLite 数据库
db = SqliteDb(db_file="tmp/data.db")

# 使用 SQLite 数据库设置基本代理
agent = Agent(db=db)
```

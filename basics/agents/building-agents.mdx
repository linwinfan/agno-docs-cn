要构建有效的智能体，从简单开始——只需要一个模型、工具和指令。一旦这个可行，根据需要逐步添加更多功能。

关键是从明确定义的任务开始，如报告生成、数据提取、分类、摘要、知识搜索和文档处理。这些早期的成功帮助您识别什么有效、验证用户需求，并为高级系统奠定基础。

例如，这是最简单的报告生成智能体（使用 Hackernews 工具包）：

```python hackernews_agent.py lines
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.hackernews import HackerNewsTools

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    tools=[HackerNewsTools()],
    instructions="就该主题写一份报告。只输出报告。",
    markdown=True,
)
agent.print_response("趋势初创公司和产品。", stream=True)
```

## 运行您的智能体

在开发智能体时，使用 `Agent.print_response()` 方法运行它。这将在您的终端中以友好的格式打印智能体的响应。

这仅用于开发目的，不建议在生产环境中使用。在生产环境中，使用 `Agent.run()` 或 `Agent.arun()` 方法。例如：

```python
from typing import Iterator
from agno.agent import Agent, RunOutput, RunOutputEvent, RunEvent
from agno.models.anthropic import Claude
from agno.tools.hackernews import HackerNewsTools
from agno.utils.pprint import pprint_run_response

agent = Agent(
    model=Claude(id="claude-sonnet-4-5"),
    tools=[HackerNewsTools()],
    instructions="就该主题写一份报告。只输出报告。",
    markdown=True,
)

# 运行智能体并打印响应
agent.print_response("趋势初创公司和产品。")

################ 流式响应 #################
stream: Iterator[RunOutputEvent] = agent.run("趋势产品", stream=True)
for chunk in stream:
    if chunk.event == RunEvent.run_content:
        print(chunk.content)

################ 流式响应和美化打印 #################
stream: Iterator[RunOutputEvent] = agent.run("趋势产品", stream=True)
pprint_run_response(stream, markdown=True)
```

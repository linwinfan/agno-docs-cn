此示例演示如何使用 `Agent.arun()` 或 `Agent.aprint_response()` 来异步运行智能体：

<Steps>

  <Step title="创建 Python 文件">
    ```bash
    touch basic_async.py
    ```
  </Step>


  <Step title="将以下代码添加到您的 Python 文件中">
    ```python basic_async.py
    import asyncio

    from agno.agent import Agent
    from agno.models.openai import OpenAIChat
    from agno.utils.pprint import apprint_run_response

    agent = Agent(
        model=OpenAIChat(id="gpt-5-mini"),
    )

    async def basic():
        response = await agent.arun(input="给我讲个笑话。")
        print(response.content)


    async def basic_print():
        await agent.aprint_response(input="给我讲个笑话。")


    if __name__ == "__main__":
        asyncio.run(basic())
        # OR
        asyncio.run(basic_print())
    ```
  </Step>

  <Snippet file="create-venv-step.mdx" />

  <Step title="安装库">
    ```bash
    pip install -U agno openai
    ```
  </Step>

  <Step title="导出您的 OpenAI API 密钥">

    <CodeGroup>

    ```bash Mac/Linux
      export OPENAI_API_KEY="your_openai_api_key_here"
    ```

    ```bash Windows
      $Env:OPENAI_API_KEY="your_openai_api_key_here"
    ```
    </CodeGroup>
  </Step>

  <Step title="运行智能体">
    <CodeGroup>
    ```bash Mac
    python basic_async.py
    ```

    ```bash Windows
    python basic_async.py
    ```
    </CodeGroup>
  </Step>

</Steps>

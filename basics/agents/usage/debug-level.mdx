---
title: 调试级别
---

此示例演示如何为智能体设置不同的调试级别。调试级别控制显示的调试信息量，帮助您在不同详细程度上进行故障排除和理解智能体行为。

<Steps>

  <Step title="创建 Python 文件">
    ```bash
    touch debug_level.py
    ```
  </Step>


  <Step title="将以下代码添加到您的 Python 文件中">
    ```python debug_level.py
    """
    此示例展示了如何设置智能体的调试级别。

    调试级别是1到2之间的数字。

    1: 基本调试信息
    2: 详细调试信息

    默认调试级别为1。
    """

    from agno.agent.agent import Agent
    from agno.models.anthropic.claude import Claude
    from agno.tools.duckduckgo import DuckDuckGoTools

    # Basic debug information
    agent = Agent(
        model=Claude(id="claude-3-5-sonnet-20240620"),
        tools=[DuckDuckGoTools()],
        debug_mode=True,
        debug_level=1,
    )

    agent.print_response("特斯拉当前的股价是多少？")

    # 详细调试信息
    agent = Agent(
        model=Claude(id="claude-3-5-sonnet-20240620"),
        tools=[DuckDuckGoTools()],
        debug_mode=True,
        debug_level=2,
    )

    agent.print_response("苹果公司当前的股价是多少？")
    ```
  </Step>

  <Snippet file="create-venv-step.mdx" />

  <Step title="安装库">
    ```bash
    pip install -U agno anthropic ddgs
    ```
  </Step>

  <Step title="导出您的 Anthropic API 密钥">


    ```bash Windows

    ```
    </CodeGroup>
  </Step>

  <Step title="运行智能体">
    <CodeGroup>
    ```bash Mac
    python debug_level.py
    ```

    ```bash Windows
    python debug_level.py
    ```
    </CodeGroup>
  </Step>

</Steps>

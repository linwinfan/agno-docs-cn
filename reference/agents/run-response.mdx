## RunOutput 属性

| 属性          | 类型                         | 默认值                        | 描述                                                                                               |
| ------------------ | ---------------------------- | ----------------------------- | ------------------------------------------------------------------------------------------------- |
| `run_id`           | `Optional[str]`              | `None`                        | 运行 ID                                                                                           |
| `agent_id`         | `Optional[str]`              | `None`                        | 运行的代理 ID                                                                                     |
| `agent_name`       | `Optional[str]`              | `None`                        | 运行的代理名称                                                                                     |
| `session_id`       | `Optional[str]`              | `None`                        | 运行的会话 ID                                                                                     |
| `parent_run_id`    | `Optional[str]`              | `None`                        | 父运行 ID                                                                                         |
| `workflow_id`      | `Optional[str]`              | `None`                        | 如果此运行是工作流的一部分，则为工作流 ID                                                          |
| `user_id`          | `Optional[str]`              | `None`                        | 与运行关联的用户 ID                                                                               |
| `content`          | `Optional[Any]`              | `None`                        | 响应的内容                                                                                        |
| `content_type`     | `str`                        | `"str"`                       | 指定内容的数据类型                                                                                |
| `reasoning_content`| `Optional[str]`              | `None`                        | 模型生成的任何推理内容                                                                            |
| `reasoning_steps`  | `Optional[List[ReasoningStep]]` | `None`                        | 推理步骤列表                                                                                      |
| `reasoning_messages` | `Optional[List[Message]]`   | `None`                        | 推理消息列表                                                                                      |
| `model`            | `Optional[str]`              | `None`                        | 运行中使用的模型                                                                                  |
| `model_provider`   | `Optional[str]`              | `None`                        | 运行中使用的模型提供者                                                                            |
| `messages`         | `Optional[List[Message]]`    | `None`                        | 响应中包含的消息列表                                                                              |
| `metrics`          | `Optional[Metrics]`          | `None`                        | 运行的使用指标                                                                                    |
| `additional_input` | `Optional[List[Message]]`    | `None`                        | 额外的输入消息                                                                                    |
| `tools`            | `Optional[List[ToolExecution]]` | `None`                        | 工具执行列表                                                                                      |
| `images`           | `Optional[List[Image]]` | `None`                        | 附加到响应的图像列表                                                                              |
| `videos`           | `Optional[List[Video]]` | `None`                        | 附加到响应的视频列表                                                                              |
| `audio`            | `Optional[List[Audio]]` | `None`                        | 附加到响应的音频片段列表                                                                          |
| `files`            | `Optional[List[File]]` | `None`                        | 附加到响应的文件列表                                                                              |
| `response_audio`   | `Optional[Audio]`    | `None`                        | 模型的原始音频响应                                                                                |
| `input`            | `Optional[RunInput]`         | `None`                        | 来自用户的输入媒体和消息                                                                          |
| `citations`        | `Optional[Citations]`        | `None`                        | 响应中使用的任何引用                                                                              |
| `model_provider_data` | `Optional[Any]` | `None` | 模型提供者特定的元数据|
| `references`       | `Optional[List[MessageReferences]]` | `None`                        | 响应中使用的引用                                                                                  |
| `metadata`         | `Optional[Dict[str, Any]]`   | `None`                        | 与运行关联的元数据                                                                                |
| `created_at`       | `int`                        | 当前时间戳                     | 响应创建的 unix 时间戳                                                                            |
| `events`           | `Optional[List[RunOutputEvent]]` | `None`                        | 运行期间发生的事件列表                                                                            |
| `status`           | `RunStatus`                  | `RunStatus.running`           | 运行状态（running、completed、paused、cancelled、error）                                           |
| `workflow_step_id` | `Optional[str]`              | `None`                        | 工作流步骤 ID（外键关系）                                                                         |

## RunOutputEvent Types and Attributes

### Base RunOutputEvent Attributes

All events inherit from `BaseAgentRunEvent` which provides these common attributes:

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `created_at` | `int` | Current timestamp | Unix timestamp of the event creation |
| `event` | `str` | Event type value | The type of event |
| `agent_id` | `str` | `""` | ID of the agent generating the event |
| `agent_name` | `str` | `""` | Name of the agent generating the event |
| `run_id` | `Optional[str]` | `None` | ID of the current run |
| `session_id` | `Optional[str]` | `None` | ID of the current session |
| `workflow_id` | `Optional[str]` | `None` | ID of the workflow if part of workflow execution |
| `workflow_run_id` | `Optional[str]` | `None` | ID of the workflow run |
| `step_id` | `Optional[str]` | `None` | ID of the workflow step |
| `step_name` | `Optional[str]` | `None` | Name of the workflow step |
| `step_index` | `Optional[int]` | `None` | Index of the workflow step |
| `tools` | `Optional[List[ToolExecution]]` | `None` | Tools associated with this event |
| `content` | `Optional[Any]` | `None` | For backwards compatibility |

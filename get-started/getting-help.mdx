# 获取帮助

我们提供多种方式帮助您成功使用 Agno。

## 社区支持

### Discord 社区
加入我们的 [Discord 服务器](https://agno.link/discord)：
- 与其他用户交流
- 向开发团队提问
- 分享您的项目和经验

### GitHub 讨论
在 [GitHub Discussions](https://github.com/agno-agi/agno/discussions) 中：
- 提出功能请求
- 参与技术讨论
- 分享最佳实践

## 文档资源

### 搜索文档
使用文档顶部的搜索功能快速找到答案。

### 示例代码
查看 [示例库](/examples/use-cases/overview) 学习实际应用。

### API 参考
查阅 [API 文档](/reference-api/overview) 了解详细信息。

## 报告问题

### Bug 报告
如果您发现了 bug，请在 [GitHub Issues](https://github.com/agno-agi/agno/issues) 中报告：

1. 检查是否已有相关 issue
2. 使用 bug 报告模板
3. 提供完整的错误信息和复现步骤

### 功能请求
想要新功能？请：
1. 检查是否已有相关讨论
2. 详细描述用例和需求
3. 说明为什么这个功能很重要

## 贡献

想要为 Agno 做贡献？查看我们的[贡献指南](/contributing)了解：
- 如何设置开发环境
- 代码贡献流程
- 文档贡献方式

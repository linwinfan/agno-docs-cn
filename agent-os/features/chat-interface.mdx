## 概述

AgentOS 聊天是您与 AI 系统进行日常工作的核心界面。在一个屏幕上，您可以：

- 与单个智能体聊天
- 与智能体团队协作
- 触发和监控工作流
- 查看会话、知识、记忆和指标

它的设计让您感到熟悉——输入消息、附加文件、获得实时的流式响应。每个智能体、团队和工作流都维护自己的上下文，因此您可以在任务之间切换而不会丢失进度。

## 聊天界面

### 与智能体聊天

- 从右侧面板选择一个智能体
- 提出问题，如"您可以访问哪些工具？"
- 智能体保持自己的历史记录、工具和指令；切换智能体不会混合上下文。

- 切换到工作流模式并选择一个工作流
- 工作流按预定义的步骤执行任务，每个步骤可能涉及不同的智能体或团队

## 聊天功能

### 消息输入

#### 基本消息

```text
您能帮我分析这份销售数据吗？
```

#### 文件附件

- 点击附件按钮 📎 上传文件
- 支持的格式：PDF、Word、Excel、图片、文本文件
- 智能体可以读取和分析附件内容

#### 代码块

使用三个反引号创建代码块：

````
```python
def analyze_data(data):
    # 分析数据的代码
    return result
```
```

#### 格式化文本

支持 Markdown 格式：
- **粗体文本**
- *斜体文本*
- `代码`
- [链接](https://example.com)

### 响应流式显示

- **实时响应**: 智能体的回答会实时流式显示
- **打字效果**: 模拟真实的打字体验
- **中断和重新生成**: 可以随时中断响应并重新生成

### 上下文管理

#### 对话历史

- 自动保存每段对话
- 可以查看历史消息
- 支持搜索历史记录

#### 会话切换

- 在不同会话之间无缝切换
- 每个会话保持独立的上下文
- 支持会话命名和分类

#### 记忆集成

- 智能体可以记住用户偏好
- 跨会话保持重要信息
- 支持手动添加记忆

## 故障排除

### 常见问题

#### 连接问题

**问题**: 无法连接到智能体
**解决方案**:
1. 检查网络连接
2. 验证 API 密钥
3. 重启应用程序

#### 响应延迟

**问题**: 智能体响应缓慢
**解决方案**:
1. 检查服务器状态
2. 清理浏览器缓存
3. 减少并发请求

#### 文件上传失败

**问题**: 无法上传文件
**解决方案**:
1. 检查文件格式支持
2. 验证文件大小限制
3. 尝试压缩文件

## 最佳实践

### 1. 有效提问

- 提供清晰的上下文
- 明确指定期望的输出格式
- 使用具体的例子说明需求

### 2. 会话管理

- 定期清理不必要的会话
- 使用有意义的会话名称
- 及时归档重要对话

### 3. 团队协作

- 建立清晰的沟通规范
- 定期同步项目进展
- 有效利用工具和自动化

### 4. 安全使用

- 定期更新密码
- 启用双重认证
- 注意敏感信息保护

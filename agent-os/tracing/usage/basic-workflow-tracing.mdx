此示例展示如何在 AgentOS 中追踪工作流执行过程。

<Steps>

  <Step title="创建 Python 文件">
    ```bash
    touch basic_workflow_tracing.py
    ```
  </Step>

  <Step title="添加工作流代码">
    ```python basic_workflow_tracing.py
    from agno.workflow import Workflow
    from agno.db.sqlite import SqliteDb
    from agno.models.openai import OpenAIChat
    from agno.os import AgentOS

    # 设置数据库
    db = SqliteDb(db_file="tmp/traces.db")

    # 创建工作流
    workflow = Workflow(
        name="数据分析工作流",
        model=OpenAIChat(id="gpt-4"),
        instructions="执行数据分析流程。",
        db=db
    )

    # 设置启用了追踪的 AgentOS
    agent_os = AgentOS(
        description="工作流追踪示例",
        workflows=[workflow],
        tracing=True,
    )
    app = agent_os.get_app()

    if __name__ == "__main__":
        agent_os.serve(app="basic_workflow_tracing:app", reload=True)
    ```
  </Step>

  <Step title="运行并查看追踪">
    在 [AgentOS UI](https://os.agno.com/traces) 中查看工作流执行追踪。
  </Step>

</Steps>

## 工作流追踪数据

工作流追踪包括：
- 工作流执行追踪
- 步骤执行追踪
- 数据流追踪
- 状态转换追踪
